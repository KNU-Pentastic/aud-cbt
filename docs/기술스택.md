# AUD CBT v3.0 — 개발자별 기술 스택

## 백엔드 개발자

- **언어**: Python 3.11+
- **프레임워크**: FastAPI
- **ASGI 서버**: Uvicorn
- **데이터베이스**: PostgreSQL
- **ORM**: SQLAlchemy 2.0 (또는 SQLModel)
- **마이그레이션**: Alembic
- **캐시·세션**: Redis
- **검증**: Pydantic v2
- **인증**: PyJWT, passlib (PIN 해싱)
- **LLM**: Anthropic Python SDK
- **SSE**: FastAPI `StreamingResponse` + `sse-starlette`
- **테스트**: pytest, httpx
- **컨테이너**: Docker, docker-compose
- **호스팅**: Railway (또는 Render)
- **API 스펙**: openapi.yaml 정본, FastAPI 자동 생성 Swagger UI
- **코드 생성 도구**: datamodel-code-generator (openapi.yaml → Pydantic 모델)

---

## 환자 앱 개발자

- **언어**: TypeScript
- **프레임워크**: React Native + Expo (SDK 최신)
- **상태 관리**: Zustand (가볍고 학습 곡선 낮음)
- **데이터 페칭**: TanStack Query (React Query)
- **HTTP 클라이언트**: openapi-fetch (openapi.yaml에서 타입 자동 생성)
- **SSE**: react-native-sse
- **네비게이션**: Expo Router (파일 기반 라우팅)
- **스타일링**: NativeWind (Tailwind for React Native)
- **폼**: React Hook Form + Zod
- **로컬 저장**: expo-secure-store (Access Token 저장)
- **알림**: expo-notifications (일일 체크인 알림)
- **빌드·배포**: EAS Build, EAS Submit
- **개발 환경**: Expo Go 앱 (와이파이 라이브 미리보기)
- **타입 생성**: openapi-typescript (openapi.yaml → TS 타입)
